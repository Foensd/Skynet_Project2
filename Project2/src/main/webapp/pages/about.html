About

Town of Revature is an entertainment based web application in which the goal is to keep your job. A rival company has sent hackers to infiltrate
Revature with the intent of firing all it's employees! The amount of Hackers is unknown so better watch your back!


Goals

  • To create an entertainment web application which can be played in real time
  • Users need to have a way to talk to each other
  • User actions need to be consistent
  • Users need to have fun !

This web application was designed in the span of a week.
